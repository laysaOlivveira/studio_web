O Studio Funcional Adriano Oliveira é um espaço especializado em treinamento funcional e musculação personalizada. Atualmente, a gestão é feita de forma manual ou com ferramentas desconectadas, o que gera retrabalho, falta de acompanhamento dos alunos e baixa automação na comunicação e nos pagamentos.

O objetivo deste projeto é desenvolver uma plataforma web responsiva e moderna que centralize todas as operações do studio, oferecendo uma experiência integrada para alunos e administrador, com foco em usabilidade, performance e automação.

Público-Alvo

Alunos de musculação

Alunos de treino funcional

Alunos que praticam ambas as modalidades

Administrador do studio

Funcionalidades Principais

Gestão de Usuários

Cadastro de alunos por modalidade (funcional, musculação ou ambos)

Perfil com status: ativo, inadimplente ou cancelado

Histórico completo por aluno

Gestão de Treinos (Musculação)

Envio de treinos individuais com imagens (banco de exercícios)

Solicitação de troca de treino liberada após 40 dias

Histórico de treinos anteriores

Gestão de Turmas (Funcional)

Organização por turmas com limite de vagas

Painel de avisos por turma

Avaliação Física

Registro de medidas, peso, altura, IMC

Histórico de evolução com comparativo automático

Avaliação mensal obrigatória

Sistema Financeiro

Integração com API de pagamentos do Banco do Brasil

Pagamentos via PIX e cartão de crédito

Fluxo de caixa com entradas automáticas e saídas manuais

Relatórios financeiros por aluno, modalidade e período (exportação em PDF)

Comunicação via WhatsApp

Notificações automáticas: matrícula, treino publicado, vencimento, pagamento

Link direto para contato no perfil do aluno

Painel Administrativo

Dashboard com métricas: total de alunos, inadimplentes, receita mensal, próximas avaliações

Gestão de turmas, treinos, exercícios, pagamentos e avisos

Painel de Avisos

Avisos gerais, por turma ou individuais

Notificação automática integrada

Histórico do Aluno

Pagamentos, treinos, avaliações, evolução corporal e feedbacks

Identidade Visual

Cores principais: Branco, Vermelho e Preto

Estilo: Moderno, minimalista, esportivo e de fácil leitura

Logos: Incluir logotipo do Studio Funcional e, se houver, da marca pessoal Adriano Oliveira

Requisitos Técnicos (para referência do Dev/UI)

Front-end: React + CSS (módulos ou styled-components)

Abordagem: Mobile First, totalmente responsiva (Desktop, Tablet, Smartphone)

Hospedagem: Vercel (com SSL)

Banco de Dados: MySQL

Segurança: Controle por perfil (Admin / Aluno), criptografia de senha, proteção contra SQL Injection e XSS

Fluxos Estratégicos para o UX Design

Cadastro: Escolha de modalidade → Pagamento → Confirmação → Liberação de acesso

Treino: Admin publica → Aluno recebe notificação → Botão de solicitar troca liberado após 40 dias

Financeiro: Pagamento via Banco do Brasil → Confirmação automática → Registro no fluxo de caixa

Avaliação: Agendamento → Registro de medidas → Comparativo gerado automaticamente
